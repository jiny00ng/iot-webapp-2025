본 메일은 소중한 고객님의 개인정보보호를 위하여 암호화하여 발송한 메일 입니다.

고객님의 주민번호 앞 6자리(생년월일), 법인/개인 사업자인 경우 사업자 번호 10자리를

비밀번호 입력창에 입력하시면, 메일의 원문을 보실 수 있습니다.

차단된 콘텐츠를 허용해 주시기 바랍니다.

본 보안메일은 멀티브라우저 및 멀티OS 환경을 지원 합니다.
PC 환경에서는 Windows와 Mac의 다양한 브라우저를 지원하며
모바일 환경에서는 Android와 iOS를 지원합니다.

◈ 모바일 환경


  • 1. Android
      가. 모바일 브라우저 이용
         1) 호환 브라우저에서는 별도의 앱 설치 없이 보안메일 확인이 가능합니다.
           - 호환 브라우저 : Firefox, Opera
           - 그 외 브라우저 : 하단의 "보안메일 전용 앱 설치 바로가기"를 클릭하여 설치

  •  나. 모바일 앱 이용
         1) 하단의 "보안메일 전용 앱 설치 바로가기"를 클릭하여 보안앱을 설치합니다.
         2) 첨부파일을 클릭 후 작업을 수행할 때 사용하는 어플리케이션 목록에서
             설치한 XecureExpressForMobile 앱을 선택합니다.

    Android 보안메일 전용 앱 설치 바로가기

  • 2. iOS
      iOS에서는 별도 보안 소프트웨어 설치 없이 보안메일 확인이 가능합니다.
      단, 간단한 텍스트만 확인 가능한 iOS 기본 메일 앱의 경우
      '보안메일 전용 앱' 설치가 필요 합니다.

      가. iOS 기본 메일 앱 보안메일 확인 방법
         1) 하단의 "iOS 보안메일 전용 앱 설치 바로가기" 를 클릭하여 보안앱을 설치합니다.
         2) 오른쪽 상단의 앱 리스트 클릭합니다.
         3) 설치한 XecureExpressForMobile 앱을 선택합니다.

    iOS 보안메일 전용 앱 설치 바로가기

◈ PC 환경


보안 메일의 첨부파일을 저장 후 실행하여 주시기 바랍니다.

차단된 콘텐츠 허용 방법

  1. 브라우저 상단의 차단 메시지 클릭

  2. 차단된 콘텐츠 허용 클릭

  3. 해당 팝업이 뜨면 '예' 클릭

  4. 차단을 해지하면, 보안메일 본인인증 팝업창 정상적으로 구동됨
